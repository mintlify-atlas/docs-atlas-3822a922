## What you can build

The CleverTap Android SDK gives your app a full suite of engagement and analytics capabilities:

- **Analytics** — Track events, user profiles, screen views, and purchase funnels
- **Push notifications** — Deliver FCM, HMS (Huawei), and Xiaomi push messages with custom channels and rich templates
- **In-app messages** — Show banners, interstitials, half-interstitials, and custom code templates triggered by user behavior
- **App Inbox** — A persistent in-app message center for promotions and updates
- **Remote Variables** — Run A/B tests and control feature flags from the CleverTap dashboard
- **Geofencing** — Trigger campaigns when users enter or exit defined geographic zones
- **RenderMax** — Improve push delivery rates on battery-optimized devices

## Requirements

- Android API level 19 (KitKat) or higher
- Java 8 or higher
- Kotlin 1.5 or higher (for Kotlin examples)
- A CleverTap account — [sign up here](https://clevertap.com/live-product-demo/)
